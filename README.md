# GPCO 400 Midterm Pre-Reading Analysis

This repository contains source readings, review materials, and written analysis artifacts for the GPCO 400 Policy Making Processes midterm.

## Purpose

The workspace is organized to support:

- reviewing assigned readings and review slides,
- referencing prior-year exam/rubric examples,
- drafting theory-to-reading application responses with in-text citations.

## Repository Contents

### Core Readings

- `Reading A.pdf` - Primary reading A (permitting reform letter).
- `Reading B.pdf` - Primary reading B (Venezuela/democracy analysis).
- `Reading A.txt` - Extracted text version of reading A.
- `Reading B.txt` - Extracted text version of reading B.

### Midterm Materials

- `PMP 2026 Midterm review.pdf` - Current review slide deck.
- `PMP 2026 Midterm review.txt` - Extracted text version of the review deck.

### Prior-Year Reference Materials

- `OLD - Midterm 2025.pdf` - Prior-year midterm prompt.
- `OLD - Midterm 2025.txt` - Extracted text version of the prior-year prompt.
- `OLD - PMP Midterm Rubric 2025.pdf` - Prior-year grading rubric.
- `OLD - PMP Midterm Rubric 2025.txt` - Extracted text version of the rubric.

### Analysis Output

- `PMP_2026_midterm_theory_application_with_intext_citations.md` - Consolidated theory application draft using quote-based in-text citations.

## Suggested Workflow

1. Review `PMP 2026 Midterm review.pdf` for topic coverage and exam structure.
2. Read `Reading A.pdf` and `Reading B.pdf` for primary source content.
3. Use the `.txt` versions for quick searching and citation line checks.
4. Draft and revise arguments in `PMP_2026_midterm_theory_application_with_intext_citations.md`.

## Notes

- No build or runtime setup is required; this is a document-based repository.
- Filenames are intentionally descriptive (including spaces) to match source materials.
